# lab3-Gas proceses

 Базовый класс - AbstractProcess, от него наследуются IsoBarProcess и IsoChorProcess.
 В абстрактном процессе меняется как объем, так и давление, а работа рассчитывается по общей формуле,
 в изобарном постоянное давление, в изохорном - постоянный объем, и из-за этого упрощаются
 формулы для работы, переопределенные в производных классах.
 
 Также для производных классов переопределен вывод: например, для изобарного процесса не имеет
 смысла выводить конечное давление, т.к. оно не изменяется
